# ARIAN DESIGNS

A clean, modern portfolio website for social media visuals, posters, and digital design.

## Personal Portfolio Website

A responsive portfolio website built using Jekyll, HTML, CSS, and JavaScript.

### Features
- Clean hero section with parallax effect
- Responsive About, Skills, and Projects sections
- Interactive cards with 3D hover tilt
- GitHub-style footer layout
- Brand-colored social icons
- Smooth anchor navigation
- Deployed on GitHub Pages
